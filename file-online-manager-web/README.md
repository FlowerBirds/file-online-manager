# file-online-manager-web

## 安装
切换npm源
```
npm config set registry https://registry.npm.taobao.org
```
初始化项目
```
npm install
```

### 运行
运行前端之前，先运行后端
```
npm run serve
```

### 构建
会在工程的同级目录下创建static目录
```
npm run build
```

### 运行代码校验
```
npm run lint
```
